## ============================================
## HOW TO RUN THE FRAMEWORK
## ============================================

### OPTION 1: Using the batch script (Recommended for Windows)
1. Double-click run-tests.bat
2. Select option from menu
3. View reports in target/cucumber-reports/

### OPTION 2: Using Maven commands
```bash
# Install dependencies first
mvn clean install -DskipTests

# Run all tests
mvn clean test

# Run specific test by tag
mvn test -Dcucumber.filter.tags="@CreateCustomer"
mvn test -Dcucumber.filter.tags="@SearchCustomer"
mvn test -Dcucumber.filter.tags="@EndToEnd"

# Generate Excel test case document
mvn exec:java -Dexec.mainClass="com.example.utils.ExcelTestCaseGenerator"
```

### OPTION 3: From IDE (Eclipse/IntelliJ)
1. Right-click on TestRunner.java
2. Select "Run As" > "JUnit Test"
3. View results in IDE test runner
